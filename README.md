# Fexus Games Unity Dev Test Task

## Controls

- **W / Up Arrow**: Move canon up
- **S / Down Arrow**: Move canon down
- **A / Left Arrow**: Move canon left
- **D / Right Arrow**: Move canon right
- **Space** / **Mouse Click**: Shoot
